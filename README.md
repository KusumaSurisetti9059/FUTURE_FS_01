Personal Portfolio Website

This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript.
The website was created as part of the Future Interns Full Stack Web Development Internship Task.



Project Objective

The objective of this project is to build a professional online portfolio to showcase my skills, projects, resume, and contact information in a modern and user-friendly manner.



Features

- Responsive Design
- Modern User Interface
- Interactive Navigation Bar
- Skills Showcase Section
- Projects Section
- Contact Form Functionality
- Resume Download Feature
- Hover Effects and Animations
- Mobile-Friendly Layout

Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages 

What I Learned

- Frontend Web Development
- Responsive Web Design
- CSS new tags & Layout Design
- JavaScript Interactivity
- Website Deployment
